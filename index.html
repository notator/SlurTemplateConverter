Slur Template Converter

This utility converts slur templates contained in a local SVG file to proper, pointed slurs.
The resulting SVG file is automatically saved in the Downloads folder using the original
file name prefixed by “converted_”.

There are two classes of slur template:
  • shortSlurTemplate: A simple, two-point path
  • longSlurTemplate: A path having two end-points enclosing one or more tangent points.

Each replacement slur:
  • has the class: slur
  • is added to the SVG file next to its template, before deleting the template.
